# TiDB 3.0.13 Release Notes

Release date: April 22, 2020

TiDB version: 3.0.13

TiUP version: 3.0.13

## Bug Fixes

+ TiDB

    - Fix the issue that the `INSERT ... ON DUPLICATE UPDATE` statement might be executed incorrectly within a transaction when users need to insert multiple rows of duplicate data because `MemBuffer` is not checked [#16690](https://github.com/pingcap/tidb/pull/16690)

+ TiKV

    - Fix the issue that the system might get stuck and the service is unavailable if `Region Merge` is executed repeatedly [#7612](https://github.com/tikv/tikv/pull/7612)
