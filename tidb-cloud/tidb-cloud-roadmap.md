# TiDB Cloudロードマップ {#tidb-cloud-roadmap}

TiDB Cloudロードマップでは、近い将来に予定されている内容が紹介されるため、新機能や改善点を事前に確認したり、進捗状況を追跡したり、途中の主要なマイルストーンを把握したりできます。開発の過程で、このロードマップはユーザーのニーズ、フィードバック、および当社の評価に基づいて変更される可能性があります。

✅: 機能または改善はすでにTiDB Cloudで利用可能です。

> **セーフハーバー声明:**
>
> 当社の文書、ロードマップ、ブログ、Web サイト、プレス リリース、または公式声明で説明または参照されている、現在利用できない未リリースの機能 (「未リリースの機能」) は、当社の裁量により変更される場合があり、計画どおりに提供されないか、まったく提供されない可能性があります。お客様は、購入の決定が現在利用可能な関数のみに基づいて行われること、および別途記載がない限り、PingCAP は契約上の合意の一環として前述の未リリースの機能を提供する義務を負わないことを承認します。
