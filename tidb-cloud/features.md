# 特徴 {#features}

このドキュメントには、 TiDB Cloud Starter、Essential、Dedicated などのさまざまなTiDB Cloudプランの機能サポート ステータスが記載されています。

> **ヒント：**
>
> [TiDB Cloudスターター](/tidb-cloud/select-cluster-tier.md#starter)はTiDB Cloudを使い始めるのに最適な方法です。さらに、 [TiDB プレイグラウンド](https://play.tidbcloud.com/?utm_source=docs&#x26;utm_medium=tidb_cloud_quick_start)でTiDB Cloud の機能を試すこともできます。

-   ✅: この機能は**一般公開されている**か、**パブリック プレビュー**段階です。
-   🔒: この機能は**プライベートプレビュー**段階です。
-   🚧: この機能は**現在開発中**です。
-   ❌: この機能は**現在利用できません**。
