# TiDB Cloud 发布记录

[TiDB Cloud](https://www.pingcap.com/tidb/cloud/) 是一款全托管的数据库即服务（DBaaS），将开源的混合事务和分析处理（HTAP）数据库 [TiDB](https://docs.pingcap.com/tidb/stable/overview) 部署到云端。TiDB Cloud 提供了一种简单便捷的方式来部署和管理数据库，使用户能够将更多精力专注于应用程序本身，而无需处理复杂的数据库运维工作。本文档概述了 TiDB Cloud 的发布记录和维护通知相关内容。

## 发布说明

TiDB Cloud 发布说明提供 TiDB Cloud 发布的新功能和改进的相关信息。详细的发布说明，请参阅 [TiDB Cloud 发布说明](/tidb-cloud/releases/tidb-cloud-release-notes.md)。

## 维护通知

TiDB Cloud 维护通知提供有关计划维护活动的信息，这些维护活动可能会对你的 TiDB Cloud 服务产生影响。
