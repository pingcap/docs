# [2024-04-18] TiDB Cloud Data Migration (DM) 功能维护通知

本通知描述了你需要了解的 2024 年 4 月 18 日 TiDB Cloud Dedicated [Data Migration (DM) 功能](/tidb-cloud/migrate-from-mysql-using-data-migration.md) 维护的相关细节。

## 维护窗口

- 开始时间：2024-04-18 08:00 (UTC+0)
- 结束时间：2024-04-18 09:00 (UTC+0)
- 持续时长：1 小时

## 影响

在维护窗口期间，以下区域的 TiDB Cloud Dedicated 集群的 DM 功能将进行升级：

- 云服务商：AWS，区域：东京 (ap-northeast-1)

升级期间，你可以正常使用 TiDB 集群和 DM 控制台的各项功能。DM 任务也会持续运行，不会中断。

## 完成与恢复

维护成功完成后，受影响的功能将恢复，为你带来更好的体验。
