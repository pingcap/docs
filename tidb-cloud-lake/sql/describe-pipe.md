# DESCRIBE PIPE

Shows the properties of a pipe.

## Syntax

```sql
DESCRIBE PIPE <name>
```

`DESC PIPE <name>` is accepted as a synonym.

## Example

```sql
DESCRIBE PIPE my_pipe;
```
