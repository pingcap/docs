# Notification

This page provides a comprehensive overview of Notification operations in {{{ .lake }}}, organized by functionality for easy reference.

## Notification Management

| Command | Description |
|---------|-------------|
| [CREATE NOTIFICATION](/tidb-cloud-lake/sql/create-notification-integration.md) | Creates a new notification integration for event alerts |
| [ALTER NOTIFICATION](/tidb-cloud-lake/sql/alter-notification-integration.md) | Modifies an existing notification integration |
| [DROP NOTIFICATION](/tidb-cloud-lake/sql/drop-notification-integration.md) | Removes a notification integration |

## Notification Information

| Command | Description |
|---------|-------------|
| [DESCRIBE NOTIFICATION](/tidb-cloud-lake/sql/describe-notification-integration.md) | Shows the properties of a notification integration |

> **Note:**
>
> Notifications in {{{ .lake }}} allow you to configure integrations with external services like email or Slack to receive alerts about database events and operations.
