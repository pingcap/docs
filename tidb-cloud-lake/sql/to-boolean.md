# TO_BOOLEAN

Converts a value to BOOLEAN data type.

## Syntax

```sql
TO_BOOLEAN( <expr> )
```

## Examples

```sql
SELECT TO_BOOLEAN('true');

┌────────────────────┐
│ to_boolean('true') │
├────────────────────┤
│ true               │
└────────────────────┘
```
