---
Returns the value of π as a floating-point value.

## Syntax

```sql
PI()
```

## Examples

```sql
SELECT PI();

┌───────────────────┐
│        pi()       │
├───────────────────┤
│ 3.141592653589793 │
└───────────────────┘
```
