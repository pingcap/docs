# CURRENT_CATALOG

Returns the name of the catalog currently in use for the session.

## Syntax

```sql
CURRENT_CATALOG()
```

## Examples

```sql
SELECT CURRENT_CATALOG();

┌───────────────────┐
│ current_catalog() │
├───────────────────┤
│ default           │
└───────────────────┘
```
