This page provides a comprehensive overview of aggregating index operations in Databend, organized by functionality for easy reference.

## Aggregating Index Management

| Command | Description |
|---------|-------------|
| [CREATE AGGREGATING INDEX](/tidb-cloud-lake/sql/create-aggregating-index.md) | Creates a new aggregating index for a table |
| [DROP AGGREGATING INDEX](/tidb-cloud-lake/sql/drop-aggregating-index.md) | Removes an aggregating index |
| [REFRESH AGGREGATING INDEX](/tidb-cloud-lake/sql/refresh-aggregating-index.md) | Updates an aggregating index with the latest data |
