---
title: RADIANS
summary: Returns the argument x, converted from degrees to radians.
---
Returns the argument `x`, converted from degrees to radians.

## Syntax

```sql
RADIANS( <x> )
```

## Examples

```sql
SELECT RADIANS(90);

┌────────────────────┐
│     radians(90)    │
├────────────────────┤
│ 1.5707963267948966 │
└────────────────────┘
```
