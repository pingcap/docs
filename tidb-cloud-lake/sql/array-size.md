# ARRAY_SIZE

Returns the length of an array, counting `NULL` elements.

Alias: `ARRAY_LENGTH`

## Syntax

```sql
ARRAY_SIZE(<array>)
```

## Return Type

`BIGINT`

## Examples

```sql
SELECT ARRAY_SIZE([1, 2, 3]) AS size_plain;
```
