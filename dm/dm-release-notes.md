# TiDB Data Migrationリリースノート {#tidb-data-migration-release-notes}

DM v5.4以降、TiDB Data Migrationのリリースノートは、同バージョンのTiDBリリースノートに統合されました。

-   v5.4以降のバージョンのDMリリースノートを読むには、対応するTiDBリリースノートのDM関連コンテンツを参照してください。
-   DM v5.3.0またはそれ以前のバージョンのリリースノートを読むには、以下のリンクを参照してください。
